# Android中MVP插件，快捷生成MVP所需的类和方法
### 1.安装
首先，你需要在你的Android Studio上安装此插件MVPPlug.
![在这里插入图片描述](https://raw.githubusercontent.com/CnRiven/MVPPlug/master/resources/icons/1.png)
### 2.使用
创建MVP时，只需要输入Activity的名字即可.
![在这里插入图片描述](https://raw.githubusercontent.com/CnRiven/MVPPlug/master/resources/icons/2.png)
### 3.提示
请根据自己的项目架构进行调整，并非适用所有的项目。
另：此插件是结合Dagger2使用的，请对创建的Presenter添加@Inject注解。
